
## Step 1 — Clone Repository

```bash
git clone https://github.com/dummy-account-100/sakthi-production
cd sakthi
```

---

## Step 2 — Install Docker Desktop

Ensure Docker Desktop is:

* Installed
* Running
* WSL2 enabled (if on Windows)

Verify:

```bash
docker --version
docker compose version
```

---

## Step 3 — Build Containers (No Cache)

From project root:

```bash
docker compose build --no-cache
```

---

## Step 4 — Start Containers


```bash
docker compose up -d
```

---

## Step 5 — Create Database 


### 5.1 Enter MSSQL container

```bash
docker ps
```

Find the container name, then:



```bash
docker exec -it sakthi-db-1 /opt/mssql-tools18/bin/sqlcmd -S localhost -U sa -P "AppUser@123" -C -No
```

### 5.2 Create Database

Inside sqlcmd:

CREATE DATABASE FormDb1;
GO;

type "exit" to leave the container shell

then copy the file from the mssql/init.sql and paste using this cmd in the root directory:

docker exec -i sakthi-db-1 /opt/mssql-tools18/bin/sqlcmd -S localhost -U sa -P "AppUser@123" -C -No < mssql/init.sql

## Step 6 — Access Application

Open browser:

```
http://localhost
```
